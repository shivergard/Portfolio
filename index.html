About me

I like building websites and making cool little things to spread around the internet. I started working in IT as a web designer, but since I found the tools lacking for designing responsive websites, animation effects and lots of other stuff, I started to code and shifted to front end development. The primary things that I write are HTML, CSS, JavaScript and jQuery. Recently I have been working as a programmer involved in web development spending my 40 h workweek in an office, building storefronts, web pages, landing pages and adding custom functionality to existing stuff. I changed from design to programming when cool concepts like meaningful markup, responsive design and the importance of typography became mainstream on making the web a better place for users.
